- App designed for portrait mode only
- App can handle duplicate names for tasks and child names, but they will be considered as separate objects
- Button for choosing nobody's turn is in the toolbar for flip coin. It's the button on the right corner
- Once nobody's turn is clicked the only way to go back to the child order is to flip the coin or go back
  to the main menu and go back into Flip A Coin.
